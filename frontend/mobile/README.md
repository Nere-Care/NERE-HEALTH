# Mobile

Ce dossier contient le squelette de l’application mobile Flutter du projet NERE.

Structure :
- `pubspec.yaml` : configuration Flutter
- `lib/main.dart` : point d’entrée de l’application
- `analysis_options.yaml` : règles d’analyse Dart
- `test/widget_test.dart` : test de base

Instructions :
1. Installez Flutter sur votre machine.
2. Depuis `frontend/mobile`, exécutez `flutter pub get`.
3. Lancez l’application avec `flutter run`.

Ce dossier est maintenant prêt à être développé comme une application mobile native.
